# SRV Matriculation School

A modern, professional web application built for SRV Matriculation School, Uppilliapuram. 

## Tech Stack
- React
- Vite
- Tailwind CSS (v4)
- Framer Motion
- Lucide React

## Development
Run the local development server:
```sh
npm run dev
```
